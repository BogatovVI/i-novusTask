## Тестовое задание по генерации номеров.
```
Добавил query параметр idUser для хранения состояния последнего номера пользователя. Можно было хранить в кеше для быстрого доступа.
Для поиска данных лучше всего использовать АВЛ-деревья с логарифмической сложностью, поэтому использовал TreeSet.
HashMap для хранения последнего номера пользователя. Класс с хранением номеров и пользователей нарушает принцип SOLID, логику нужно было разделять.
Реализация через массив констант тоже костыль, нужно использовать коды символов, моя реализация упадет при добавления нового символа.
```
